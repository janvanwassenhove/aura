# AURA brain vault

Every `.md` file here is one skill the owner taught the assistant.
This folder is a plain markdown vault — open it in Obsidian if you
like: `[[person-id]]` / `[[skill-name]]` links work there AND render
as clickable links in the AURA app (with backlinks on the person's
profile). Person data itself is NOT stored here — profiles stay
encrypted inside the app (see docs/user-guide.md).
